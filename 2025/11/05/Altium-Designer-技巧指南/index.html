Altium Designer 技巧指南

发表于 2025-11-05 分类于 开发工具 ， EDA ， Altium Designer 阅读次数： 本文字数： 516 阅读时长 ≈ 2 分钟

Altium Designer 技巧指南

多层级原理图快速确定网络连接关系

Tools -> Preferences -> System -> Design Insight

该栏目下包含四个可供勾选的项目，博主这里全部勾选了。

同时在Connectivity Insight Options下，可以选择层级关系显示的触发方式：

  • Mouse Hover：即鼠标放置在指定网络上，其中触发延迟可以由本页中的Mouse Hover Delay滑动条选择。

  • Alt + Double Click：即按住Alt键的同时鼠标双击指定网络这里可以根据需求自由选择，当然博主这里也是全部选上了。

1. 快捷键

1.1 栅格与尺寸相关

1.1.1 尺寸

Altium Designer 包括两种尺寸，mil和mm，其换算关系为$$1mil = 0.0254mm$$
所以，在绘制原理图时，由于不同元件的尺寸不同，需要在工作中切换不同的吸附栅格来保证元件快速对齐与设计美观。

快捷键 功能
V + U 切换单位

也可以在原理图或PCB的Properties下的General分类下点击Units的mm或者mil进行切换。

1.1.2 栅格 Grid

Altium Designer 包括三种栅格:

  • 可视栅格
    工作区上看到的网格，作用类似于坐标线，帮助用户掌握元件间的距离。

快捷键 功能
Ctrl + Shift + G 切换可视栅格尺寸

  • 吸附/捕捉栅格
    控制光标每次移动的距离，可以理解为鼠标拖动元件时，元件最小移动间距。

快捷键 功能
G 正向按顺序切换捕获栅格尺寸
Shift + G 反向按顺序切换捕获栅格尺寸

  • 电气栅格
    电气栅格的作用是在移动或放置元件时，当元件与周围电气实体的距离在电气栅格的设置范围内时，元件与电气实体会互相吸住。

快捷键 功能
Shift + E 切换电气栅格

注意：
Altium Designer 预设了几组栅格尺寸，Tools -> Preferences -> Schematic -> Design Grids在这里可以按需设定。
